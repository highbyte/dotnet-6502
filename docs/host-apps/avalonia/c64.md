# C64 in the Avalonia apps

C64-specific features in the [Avalonia Browser](browser.md) and
[Avalonia Desktop](desktop.md) apps. The two runtimes share almost all
of this; differences are called out inline as **Browser** / **Desktop** tabs or *Browser only*
notes.

For automated startup (loading programs / pasting BASIC from a URL or the command line), see the
app pages' startup-parameter reference:
[Browser URL query parameters](browser.md#url-query-parameters) /
[Desktop CLI arguments](desktop.md#cli-arguments).
