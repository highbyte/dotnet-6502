# Silk.NET

*This page is about a library, not the SilkNetNative app.*

Library: `Highbyte.DotNet6502.Impl.SilkNet`

- InputHandlers implemented with the [`Silk.NET`](https://github.com/dotnet/Silk.NET) windowing library. Can be used from a native Silk.NET application.
- Renderers implemented with Silk.NET OpenGL bindings, together with custom shaders. Can be used from a native Silk.NET application.
