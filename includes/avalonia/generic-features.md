## Default program

The example 6502 machine code that is loaded and run by default for the *Generic* computer is an
assembled version of [this 6502 assembly code](https://github.com/highbyte/dotnet-6502/blob/master/samples/Assembler/Generic/hostinteraction_scroll_text_and_cycle_colors.asm).

For the system itself, see the [Generic system overview](../../systems/generic/overview.md).
